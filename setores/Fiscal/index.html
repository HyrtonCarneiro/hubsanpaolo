Robô de Classificação (Alpha)

O que é o Robô?

O Robô de Classificação é um script inteligente desenvolvido em Python que utiliza OCR (Reconhecimento Óptico de Caracteres) para ler seus arquivos PDF e classificá-los automaticamente entre:

  • Boletos Bancários
  • NFS-e (Nota Fiscal de Serviços)
  • Notas de Débito

Requisito Obrigatório

Para funcionar, você precisa ter o Tesseract OCR instalado no seu Windows no caminho padrão: C:\Program Files\Tesseract-OCR\tesseract.exe

Baixar Script Python (.py) Ver Instruções do Executável (.exe)

Como gerar o EXE?

1

Instale o Python

Baixe e instale o Python 3.10+ do site oficial, marcando "Add Python to PATH".

2

Instale dependências

Abra o terminal e rode: pip install pymupdf pytesseract pillow pyinstaller

3

Gere o Executável

Rode no terminal: pyinstaller --onefile classificador.py

O arquivo gerado estará na pasta dist/ e funcionará em qualquer máquina Windows sem precisar de Python instalado!
